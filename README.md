# Pet Database

A Java console application that manages a simple database of pets. The program allows users to add, search, and display pet records stored in a text file.

## Features
- Add new pet records
- Store and retrieve data from files
- Display stored pets in a structured format
- Basic input validation

## Technologies
- Java
- File I/O
- Object Oriented Programming

## Purpose
This project was created as a learning exercise for working with Java classes, file storage, and basic database style operations.
